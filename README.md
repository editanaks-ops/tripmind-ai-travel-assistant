# TripMind AI Travel Assistant

## Overview

TripMind AI is a prototype AI-powered travel assistant designed to help users create personalized travel itineraries based on their interests, destination, travel style, budget, and trip duration.

The project demonstrates how AI can be used to simplify travel planning and provide tailored recommendations for individual travelers.

## Features

* Personalized travel recommendations
* Destination-based itinerary generation
* Travel style customization
* Budget-aware suggestions
* Interactive travel planning experience

## Project Goal

The goal of the project was to explore the use of AI for travel planning and user-centered recommendation systems.

## Technologies

* AI Prompt Engineering
* Large Language Models (LLM)
* Google AI Studio
* Bolt.new
* UX Design
* Travel Recommendation Logic
